# School Assessment Reports Feature

This module handles school assessment reports functionality.

## Folder Structure

```
features/school-assessment-reports/
├── actions/        # Server actions for database operations
├── components/     # React components specific to this feature
├── hooks/          # Custom React hooks
├── types/          # TypeScript type definitions
├── utils/          # Utility functions
└── README.md
```

## Usage

Import from this feature using:
```typescript
import { ... } from '@/features/school-assessment-reports/actions'
import { ... } from '@/features/school-assessment-reports/components'
```
